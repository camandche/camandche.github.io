Margin, Border, and Padding

CSS Box Model

The CSS Box Model is used to create the design and layout of web pages. The web browser renders every element as a rectangular box according to the CSS box model which is a container that contains multiple properties including borders, margin, padding and the content itself.

The Content area consists of content like text, image, or other media content. The content has padding around it, which is then surrounded by a border, which then has margins outside of it that determine the space between this element and its neighbor(s).



It's important to note that to calculate the correct width and height of an element you must also add the width and height of padding, borders and margins to the content area.

Margin

Margin is the CSS property that controls spacing between elements, it is the space between the border and the next element of your design. Margin goes around all four sides of the content and you can target and change the margin for each side.

Margins can be set as pixels, em/rem values or percentages that are relative to the width of the containing block.

The margin property allows you to set them all in one declaration. If you want a different margin length or percentage for one or more margin, it can be done so individually starting with the top value, then moving clockwise (top 20px, right 60px, bottom 30px, left 100px).

If your top/bottom values are the same, and your left/right values are the same, you can also just set two values: one for the top/bottom (20px) margins and another for left/right (30px) margins.

Padding

The Padding area is the transparent space around the content area and within the border. Its dimensions are given by the width and the height of the padding-box. Similar to margin, padding goes around all four sides of the content and you can target and change the padding for each side. Its values can be set individually or grouped together as top/bottom and right/left.
